# hubot-mqtt

Adds Conversation capabilities to Hubot

See [`src/scripts/hubot-mqtt.js`](src/scripts/hubot-mqtt.js) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-mqtt --save`

Then add **hubot-mqtt** to your `external-scripts.json`:

```json
[
  "hubot-mqtt"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
